Anleitung

Nachdem Sie eine Runde gestartet haben, leuchten auf dem Raster Kacheln farbig auf. Diese sollen Sie sich merken und durch Klicken auf dem Raster markieren. Die Reihenfolge spielt dabei keine Rolle. Für jede korrekt markierte Kachel erhalten Sie Punkte. Nachdem Sie alle Runden absolviert haben, können Sie Ihren Punktestand mit den Highscores vergleichen.
